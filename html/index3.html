JavaScript (JS) - мультипарадигменный язык программирования. Поддерживает различные стили программирования.

История

Над JavaScript работали Бренд Эйх, Марк Андрессен и Билл Джой. Первоначально язык назывался Mocha (мока), затем он был переименован в LiveScript и предназначался как для программирования на стороне клиента, так и для программирования на стороне сервера. После по маркетинговым соображениям был переименован в JavaScript. И был эксклюзивом для Netscape Navigator. Позже компания MicroSoft выпустила аналог Jscript для Internet Explorer. Позже компании прешли к общему соглашению и JavaScript одинаков для всех браузеров.

Применение

JavaScript используется для взаимодействия с пользователем через события. Выпадающее меня при наведении курсора мыши. Управление содержимым и стилями HTML элементами. Добавление анимации и графических эффектов на страницы и многое другое. Я использовал JavaScript для создания игры Змейка.

Скрипты записывают в .js файлы которые после подключают в HTML документе.
