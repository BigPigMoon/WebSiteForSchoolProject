Что такое CSS

CSS (Cascading Style Sheets - Каскадные таблицы стилей) - язык описания внешнего вида документа, написанного с использованием языка разметки HTML.

История CSS

CSS создавался с целью описания внешней структуры сайта от его внешнего вида. Такое разделение позволяло увеличить доступность документа и предоставить большую гибкость и возможность управления его представлением, а также уменьшить сложность и повторяемость в структурном содержимом. Так в 1990 году стало ясно, что нужны единые - правила по которому программисты и веб-дизайнеры проектировали бы сайты, но в 1990 году вышел HTML версии 4 и потребность в CSS отпала. После, в 1994 году возможностей HTML не хватало и Хокон Виум Ли предложил термин каскадные таблицы стилей, совместно с Бертом Босом он стал развивать CSS и в декабре 1996 года W3C приняла рекомендацию CSS1. В 2011 году W3C приняла рекомендацию CSS2. На данный момент уже есть CSS версии 3.

Синтаксис языка CSS

Перед началом работы с языком его нужно встроить в HTML документ. Это можно сделать 2 способами:

  1. С помощью тега <style>
  2. Добавив файл .css в HTML документ

Предпочтительным считается 2 способ, который я и использовал. CSS имеет простой синтаксис все стили в CSS записываются следующим способом селектор {свойство: значение} в качестве селектора выступает элемент из HTML или класс. В качестве свойство используется ключевое слово языка CSS, например, font-size для изменения размера шрифта или font-family для изменения шрифта. Значение может быть как определенная величина так и цвет. Весь код на языке CSS можно набрать в обычном блокноте и проверить в браузере.

Анимация

Анимация придает веб-странице динамичность, она оживляет ее и улучшает взаимодействие с пользователем. Создание анимации базируется на ключевых кадрах, которые позволяют автоматически воспроизводить и повторять эффекты на протяжении заданного времени, а также останавливать анимацию внутри цикла.
